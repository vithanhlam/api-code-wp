# CodeWP API for Android and iOS

This plugin provides an API for Android and iOS applications.

## 3rd Party Services

This plugin uses the following 3rd party services:

- **Expo Push Notification Service**: This service is used to send push notifications.
- Service URL: [https://exp.host/--/api/v2/push/send](https://exp.host/--/api/v2/push/send)
- Privacy Policy & Terms of Use API CODE-WP: [https://code-wp.com/chinh-sach-bao-mat/](https://code-wp.com/chinh-sach-bao-mat/)
- **Terms of Use Expo**: [Expo Terms of Use](https://expo.io/terms)
- **Privacy Policy Expo**: [Expo Privacy Policy](https://expo.io/privacy)

## Sanitization, Validation, and Escaping

All data inputs in this plugin are sanitized, validated, and escaped to prevent security vulnerabilities such as XSS and MITM attacks.


Để sử dụng các bạn đăng ký tài khoản tại [https://code-wp.com/app-code-wp/](https://code-wp.com/app-code-wp/) và bấm mua ứng dụng API CODE-WP để kết nối với website và đăng nhập sử dụng

== Installation ==

1. Tải về và upload thư mục lên /wp-content/plugins/, hoặc tìm trong plugins Wordpress trong admin tìm ứng dụng API CODE-WP kết nối ứng dụng Android - iOS.
2. Activate the plugin through the ‘Plugins’ screen in WordPress
3. API CODE-WP -> Cấu Hình API

== Frequently Asked Questions ==

= Làm thế nào có thể kết nối được với ứng dụng =

Để kết nối được với ứng dụng bạn cần tải đăng ký và mua api tại website https://code-wp.com/app-code-wp/ và sẽ nhận được mã token để thêm vào website.

= Sau khi kết nối làm sao để đăng nhập =

Sau khi kết nối thành công bạn có thể sử dụng App để truy cập, những tài khoản có quyền administrator sẽ được phép quy cập vào ứng dụng .

= Vấn đề bảo mật =

Chúng tôi sẽ không lưu trữ bất kì thông tin tài khoản và mật khẩu nào của bạn trên hệ thống của chúng tôi, bạn có thể hoàn toàn xóa nó khi không sử dụng ứng dụng nữa.
